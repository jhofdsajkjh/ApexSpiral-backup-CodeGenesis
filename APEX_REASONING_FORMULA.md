## 一、APEX 核心公式（曝光加法系统）

### 核心加法公式
```
E_v = A_v + T_v = B_v + S_v
```

| 符号 | 含义 | 单位 |
|------|------|------|
| E_v | 曝光值 (Exposure Value) | EV |
| A_v | 光圈值 (Aperture Value) | f-stop |
| T_v | 快门速度值 (Time Value) | 1/s |
| B_v | 场景亮度值 (Brightness Value) | cd/m² |
| S_v | 感光度值 (Sensitivity Value) | ISO |

### 基础曝光方程
```
A² / T = (B × S) / K
```

## 二、Reasoning Skills 融合公式

### 推理性能小公式（APEX融合版）
```
R_p = S_c + C_i = T_c + A_c
```

| 符号 | 含义 | 对应APEX |
|------|------|----------|
| R_p | 推理性能 (Reasoning Performance) | E_v (曝光值) |
| S_c | 技能编码质量 (Skill Coding) | A_v (光圈值) |
| C_i | 技能库检索精度 (Retrieval Accuracy) | T_v (快门值) |
| T_c | Token压缩率 (Token Compression) | B_v (亮度值) |
| A_c | 准确率增益 (Accuracy Gain) | S_v (感光度值) |

### 极简背诵版（论文/摘要专用）
```
R = S ⊕ C = T ⊕ A
```
**读作：R等于S并C，等于T并A**

## 三、量化数据公式

### 核心性能提升
```
ΔT = -90%    // Token减少90%
ΔA = +5.8%   // 准确率提升5.8%
Δt_time = -80%  // 时间减少80%
```

### 核心结论（一句话）
```
更少 Token + 更高准确率 = 更强推理性能
```
